# InsightHub

Enterprise Research Knowledge Assistant.

## Phase 1: Project Initialization

This repository is being implemented in phases.

### Current scope
- Repository scaffold
- Backend and frontend folder structure
- Python environment configuration placeholders
- Requirements and environment templates

### Planned stack
- FastAPI backend
- Streamlit frontend
- SQLite metadata store
- ChromaDB knowledge base
- LangGraph RAG workflow

### Local setup
1. Create and activate the `insight-hub` Conda environment with Python 3.11.15.
2. Install dependencies from `requirements.txt`.
3. Copy `.env.example` to `.env` and fill in your keys.
4. Run the backend and frontend entry points once they are added in later phases.
